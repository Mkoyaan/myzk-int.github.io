## imgディレクトリ
ファイル名はスネークケースで記述しています。
以下に例を示します。

```
about_b.svg
about_c.svg
int_t.svg
```

*_b.svg
となっているものは、ボタン用の画像です。

（例）
about_b.svg

<img src="sample/about_b.svg" width="40%">

*_c.svg
となっているものは、見出し用の画像になります。

（例）
about_c.svg

<img src="sample/about_c.svg" width="40%">

*_t.svg
となっているものは、ページトップに表示している画像です。

## sampleディレクトリ
READMEに使用している画像などのサンプルファイルを置いているディレクトリになります。
